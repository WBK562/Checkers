# Checkers Game (Warcaby) - Java Implementation

A classic desktop implementation of the Checkers board game, developed using **Java** and the **Swing** library. This project was created as part of my computer science studies at the **Lodz University of Technology (WEEIA)**.

## 🚀 Features
- **Full Game Logic:** Implementation of standard checkers rules, including mandatory captures and piece promotion.
- **Graphical User Interface (GUI):** Built with Java Swing, featuring custom-scaled graphics.
- **Move Validation:** Highlighted legal moves and prevention of illegal piece placement.
- **Queen Promotion:** Automatic promotion of pieces to "Queens" (Damki) upon reaching the opponent's back row.
- **Resource Management:** Clean project structure with externalized assets (images).

## 🛠️ Tech Stack
- **Language:** Java 17+
- **Version Control:** Git
- **IDE:** IntelliJ IDEA

## 🧠 Key Learnings
### During this project, I focused on:
- **Object-Oriented Programming (OOP): Encapsulating game logic, board state, and player actions.**
- **Event Handling: Managing mouse clicks and GUI updates in real-time.**
- **Resource Loading: Efficiently handling and scaling .png files within a Java application.**
